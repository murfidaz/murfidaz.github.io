#### 4.4.3 Install GVM

- install gvm

  - install golang versi 1.17.13

    `gvm install go1.17.13`

  - install golang versi 1.21.1

    gunakan golang versi 1.17.13 sebelum install versi 1.21.1

    `gvm use go1.17.13`

    kemudian install versi 1.21.1

    `gvm install go1.21.1`
